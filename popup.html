How to demo each Chrome extension API

01. Chrome omnibox API

Type demo in the address bar and press space or tab

02. Chrome Commands API

Press Ctrl+Shift+Y in Windows or Cmd+Shift+Y in Mac

03. Chrome Devtools API

Right click on the web page and click on inspect element. You will see a new panel with the name Demo panel has been added.

04. Chrome ttsEngine API

Type in a sentence and press start text-to-speech button

05. Chrome permissions API

Click on the following button to see the extension requesting permission to open the camera.If you can remember, this permission was not requested at the time of installation.
